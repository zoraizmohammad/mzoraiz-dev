# Mohammad Zoraiz — Portfolio

A dependency-free personal portfolio built as static HTML, CSS, and JavaScript.

## Local preview

Run from this directory:

```sh
python3 -m http.server 4174 --bind 127.0.0.1
```

Then open `http://localhost:4174`.

## Pages

- `index.html` — home
- `projects.html` — selected work
- `resume.html` — experience and skills
- `about.html` — background and activities

Changes pushed to `main` deploy through the connected Vercel project.
